Actuator Travel Time

How long the software waits in OPENING / CLOSING state before declaring the move complete. The Linak internal limit switches stop the motor automatically at both ends, so this is a display timer rather than a hard stop — but it should match your actual stroke time.

Set it slightly longer than the actual full-stroke time (e.g. if the actuator takes 8 s, set 10 s).

Stress Test — Automated Cycle

Repeatedly opens and closes the silo lid to test the actuator and relay under load. The controller waits for each move to complete before starting the next. Ensure the area around the actuator is clear before starting.

Relay Toggle Behaviour

The DR25E01 is a bistable toggle relay — each LOW pulse on T flips it to the opposite position. The software tracks which position the relay is in and only pulses when a direction change is needed.

The tracked state is saved to silo_state.json and survives reboots and service restarts. If the physical relay ever gets out of sync, use the buttons above to re-synchronise.

Hardware — GPIO Wiring

All pin numbers are BCM (Broadcom) numbering as used by gpiozero.

Pi GPIO 17→ DR25E01 T (Trigger)
Pi 5 V pin→ DR25E01 V (coil power)
Pi GND→ DR25E01 G (ground)
Relay ModuleEletechsup DR25E01 — 3A 1-ch bistable DPDT, 5 V logic
ActuatorLinak LA 121P00 12 V DC linear actuator
Relay OUT+/OUT−→ Actuator motor terminals
12 V supply→ Relay motor power input (separate from logic)
Pulse width100 ms (PULSE_MS in core/silo.py)
Travel time12 s safety ceiling (core/silo.py TRAVEL_TIME)
The relay is bistable toggle — each 100 ms pulse on T flips it to the opposite position and it holds there with no power. The software tracks which position the relay is in so it only pulses when a direction change is needed. The Linak internal limit switches stop the motor at both ends; TRAVEL_TIME is a backup.
If open/close are backwards, swap OUT+/OUT− on the actuator terminals, or set RELAY_STARTS_OPEN = True in core/silo.py.
